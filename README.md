# Scripts-and-Tools

A collection of custom Python and Bash scripts designed to automate security tasks. This repository documents my journey in automating penetration testing workflows, vulnerability research, and OSINT gathering.

## Overview
This repository serves as a centralized hub for various security utilities. Each tool is housed in its own directory with dedicated documentation to ensure clarity and ease of use.
